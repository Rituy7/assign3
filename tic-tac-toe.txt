I started off by following the official React tutorial to get the core game logic running. I set up the basic Square component and the Board layout, initially just with hardcoded values. From there, I made the squares interactive by using useState and click handlers.

The biggest part of the setup was lifting the state from the individual squares up to the Board component so I could manage turn-taking between X and O and track the overall game state. I also added the calculateWinner function to check for the standard winning lines. Finally, I refactored everything into a Game component to implement the move history. This involved using an array of board states and mapping over them to create the "time travel" buttons that let you jump back to previous moves.
